# 「无影」目录结构清单

**版本**：V1.0  
**适用范围**：无影产品方向（中台集成赋能）

---

## 1. 顶层结构

```
无影/
├── apps/                 # 可执行应用
├── docs/                 # 无影文档
├── packages/             # 无影内复用模块
├── scripts/              # 构建/测试/发布脚本
└── wu-ying/              # 现有前端原型/站点
```

---

## 2. apps 结构说明

```
apps/
└── core-service/         # 核心后端服务占位骨架
    ├── src/
    │   ├── contexts/     # DDD 有界上下文
    │   │   ├── connector-factory/
    │   │   ├── embedding-sdk/
    │   │   ├── workflow-trigger/
    │   │   ├── security-gateway/
    │   │   ├── integration-audit/
    │   │   └── developer-ecosystem/
    │   ├── domain/
    │   ├── application/
    │   ├── infrastructure/
    │   └── interfaces/
    ├── tests/
    └── README.md
```

---

## 3. 上下文说明（对应业务模块）

- **connector-factory**：连接器库、配置、同步与状态  
- **embedding-sdk**：嵌入式SDK与组件配置  
- **workflow-trigger**：事件监听、规则与编排  
- **security-gateway**：鉴权、脱敏、配额  
- **integration-audit**：审计、日志与告警  
- **developer-ecosystem**：SDK、模板、上架与生态
