# 「灵枢」目录结构清单

**版本**：V1.0  
**适用范围**：灵枢产品方向（后台智能中枢）

---

## 1. 顶层结构

```
灵枢/
├── apps/                 # 可执行应用
├── docs/                 # 灵枢文档
├── packages/             # 灵枢内复用模块
├── scripts/              # 构建/测试/发布脚本
└── ling-shu/             # 现有前端原型/站点
```

---

## 2. apps 结构说明

```
apps/
└── core-service/         # 核心后端服务占位骨架
    ├── src/
    │   ├── contexts/     # DDD 有界上下文
    │   │   ├── ontology-factory/
    │   │   ├── knowledge-ingestion/
    │   │   ├── knowledge-graph/
    │   │   ├── agent-factory/
    │   │   ├── governance/
    │   │   ├── open-platform/
    │   │   └── knowledge-production/
    │   ├── domain/
    │   ├── application/
    │   ├── infrastructure/
    │   └── interfaces/
    ├── tests/
    └── README.md
```

---

## 3. 上下文说明（对应业务模块）

- **ontology-factory**：本体建模、语义映射、冲突仲裁  
- **knowledge-ingestion**：多源接入、加工流水线、质量网关  
- **knowledge-graph**：图谱存储、复合查询、推理与算法  
- **agent-factory**：智能体编排、运行与监控  
- **governance**：审计、权限、演化治理  
- **open-platform**：API网关、事件订阅、Webhook  
- **knowledge-production**：知识生产、评审与协同
