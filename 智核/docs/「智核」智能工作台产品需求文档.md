### **「智核」智能工作台产品需求文档 (PRD)**

**文档版本**：V1.0
**产品定位**：面向企业全员的新一代智能工作平台
**核心价值**：作为“灵枢”智能中枢的能力呈现者与价值交付者，通过自然交互和智能辅助，将企业集体智慧转化为每一位员工的认知生产力，实现工作方式的范式升级。

---

### **一、 产品概述**

#### **1.1 产品愿景**
成为每位员工不可或缺的“首席认知助理”，让获取知识、应用知识、创造知识像对话一样自然，大幅提升决策质量与创新效率。

#### **1.2 核心问题与价值主张**
*   **解决的核心问题**：信息查找困难、知识与应用脱节、隐性经验无法复用、跨系统操作繁琐。
*   **提供的核心价值**：
    1.  **一个入口，解决所有**：通过自然语言对话，统一处理信息查询、任务安排、内容生成等复杂工作。
    2.  **知识主动找人**：基于工作上下文，将相关知识、案例与洞察精准推送到眼前。
    3.  **人机协同创作**：在专家熟悉的文档创作界面中，嵌入AI深度辅助，提升内容质量与结构化程度。
    4.  **智能随处可得**：通过嵌入式组件，将智能能力注入CRM、代码编辑器等各类业务工具中。

#### **1.3 核心用户角色**
| 角色 | 核心需求与痛点 | 智核核心价值 |
| :--- | :--- | :--- |
| **所有员工 (通用用户)** | 快速找到工作所需信息、答案；高效完成日常任务。 | 一个对话入口解决所有问题；获得主动、精准的知识推送。 |
| **领域专家/知识生产者** | 经验难以有效沉淀和复用；重复回答类似问题；文档创作耗时。 | 提供AI辅助创作，大幅降低知识结构化成本；一键将对话转化为知识。 |
| **管理者/分析师** | 需要快速整合信息进行决策分析；需要团队知识全景视图。 | 快速策展报告；获取基于全局知识网络的深度洞察与分析。 |
| **团队负责人** | 促进团队知识共享与传承；提升团队协作效率。 | 提供专属团队空间，自动沉淀项目知识，形成团队智慧资产。 |

### **二、 产品架构与设计原则**

#### **2.1 核心设计原则**
1.  **对话优先 (Conversation-First)**：自然语言是最高效的交互方式，覆盖绝大多数需求。
2.  **情境感知 (Context-Aware)**：系统需深度理解用户当前的工作任务、角色和所处应用环境，提供精准服务。
3.  **渐进式智能 (Progressive Intelligence)**：在保留传统文档树等熟悉视图的同时，智能引导用户向更高效的图谱视图、对话交互迁移。
4.  **灵枢能力前台化 (Frontend of LingShu)**：所有前台功能必须清晰对应并调用“灵枢”平台的一项或多项核心后台能力（如知识图谱、智能体、本体）。

#### **2.2 与“灵枢”平台的协同关系**
智核是“灵枢”智能中枢的官方前端和核心交互界面。其自身不存储或处理核心知识逻辑，所有智能行为均通过调用“灵枢”平台的API与服务实现。
*   **依赖**：智核的“智能”完全来源于“灵枢”提供的统一知识图谱、智能体引擎、本体模型与加工能力。
*   **反馈**：智核收集的用户交互、创作、反馈数据，是驱动“灵枢”知识网络和智能体持续优化演进的关键燃料。

### **三、 详细功能需求**

#### **3.1 模块一：主工作台 (智能交互中枢)**
**目标**：作为用户进入智核的默认首页和统一智能入口。

| 功能 | 需求描述 | 关联“灵枢”能力 |
| :--- | :--- | :--- |
| **自然语言对话界面** | 1. **智能问答**：用户以自然语言提问，直接返回精准答案、摘要及来源引用。<br>2. **复杂任务处理**：用户可下达复杂指令（如“为下周华东区销售会议准备一份市场趋势分析”），系统需理解意图、分解任务、协调智能体执行并汇报进展。<br>3. **多轮对话与记忆**：完整记忆对话上下文，支持深入的、连续的问题探讨。 | 知识图谱查询、智能体工厂、自然语言理解 |
| **智能摘要与任务看板** | 1. **每日智能摘要**：自动生成个性化摘要，包含待办任务、近期关联知识更新、系统推荐。<br>2. **一体化任务看板**：集中展示所有由AI创建或分配的任务（如“审阅知识草稿”、“完成项目复盘”），支持状态管理。 | 智能体工厂、用户画像、知识图谱 |
| **全局智能搜索** | 1. **融合搜索框**：在首页提供醒目搜索框，输入同时启动关键词、语义、图谱关系查询。<br>2. **智能结果聚合页**：结果页分栏清晰展示“最佳答案”、“相关文档”、“关联专家”、“过往讨论”。 | 知识图谱复合查询 |

#### **3.2 模块二：智能知识库 (核心生产与消费环境)**
**目标**：提供超越传统Wiki的智能化、结构化知识创作与管理体验。

| 功能 | 需求描述 | 关联“灵枢”能力 |
| :--- | :--- | :--- |
| **多视图浏览** | 1. **文档树视图**：提供与传统知识库类似的文件夹视图，平滑过渡用户习惯。<br>2. **知识图谱视图**：可视化展示知识网络，允许用户通过点击实体（如产品、客户）探索关联知识。<br>3. **主题/项目视图**：按业务领域、项目聚合所有相关知识资产。 | 知识图谱可视化、本体模型 |
| **AI辅助创作空间** | 1. **富文本与多模态编辑**：支持文档、表格、思维导图，并可嵌入图表、视频等。<br>2. **核心AI辅助功能**：<br>   • **内容建议**：基于当前主题，实时推荐“灵枢”图谱中的相关案例、数据片段。<br>   • **结构生成**：根据文档类型（如方案、报告）自动生成大纲。<br>   • **术语规范检查**：自动提示使用本体中的标准术语，标红未定义的专有名词。<br>3. **即时关联与图谱编辑**：编辑时侧边栏显示相关实体，支持一键插入为智能链接，直接丰富后台知识图谱。 | 知识图谱关联查询、智能体生成技能、本体论工厂 |
| **版本、状态与协作** | 1. **可视化版本对比**：清晰展示文档历史版本与差异。<br>2. **嵌入式评审**：支持对段落进行评论、@同事，讨论内容结构化保存并与知识块关联。<br>3. **状态与生效管理**：作者可直接设置知识状态（草稿/评审/发布），并指定生效范围与有效期。 | 知识图谱版本管理、权限服务 |

#### **3.3 模块三：情境知识面板 (无处不在的智能)**
**目标**：将智能从主应用延伸到用户工作的每一个具体场景中。

| 功能 | 需求描述 | 关联“灵枢”能力 |
| :--- | :--- | :--- |
| **浏览器插件/浮动窗口** | 1. **划词取义**：在任何网页或企业应用（通过“无影”套件接入）中选中文本，面板自动解释该术语，并关联内部知识。<br>2. **场景智能伴读**：当用户在特定系统（如CRM）中工作时，面板自动展示当前客户、订单、代码文件的关联知识全景（历史、风险、最佳实践）。 | 知识图谱查询、语义理解、上下文感知 |
| **快速捕获工具** | 提供浏览器插件和桌面悬浮窗，支持快速截图、添加笔记，一键保存为“知识卡片”草稿至“灵枢”待处理队列。 | 知识熔炉接入管道 |

#### **3.4 模块四：团队空间与项目协作**
**目标**：为团队协作提供智能化的项目知识管理环境。

| 功能 | 需求描述 | 关联“灵枢”能力 |
| :--- | :--- | :--- |
| **专属团队空间** | 1. **空间门户**：为每个团队或项目创建独立空间，聚合所有相关文档、对话、任务、成员。<br>2. **自动知识沉淀**：自动汇集空间内产生的会议纪要、决策、文件，形成项目知识图谱子集。<br>3. **团队专家发现**：基于贡献与互动数据，可视化展示团队内各领域“隐性专家”。 | 知识图谱子图、用户贡献分析 |

#### **3.5 模块五：智能体市场与应用**
**目标**：让用户能够发现、定制并使用解决特定场景问题的AI助手。

| 功能 | 需求描述 | 关联“灵枢”能力 |
| :--- | :--- | :--- |
| **智能体商店** | 1. **发现与订阅**：浏览由“灵枢-智能体工厂”发布和认证的各类智能体（如“合同审查官”、“代码评审助手”、“销售话术教练”），可一键订阅。<br>2. **评价与反馈**：用户可对使用过的智能体进行评分和评价。 | 智能体工厂 |
| **我的智能体工作台** | 1. **个性化配置**：对订阅的智能体进行简单配置（如设定其关注的业务范围）。<br>2. **智能体协同**：可在一个任务中同时调用多个智能体协同工作。 | 智能体运行管理 |

#### **3.6 模块六：个人中心与管理后台**
**目标**：提供个人知识管理、贡献激励及团队级管理能力。

| 功能 | 需求描述 | 关联“灵枢”能力 |
| :--- | :--- | :--- |
| **个人知识主页** | 1. **我的贡献**：可视化展示个人创建、编辑、评审的知识条目的影响力数据（浏览量、引用量）。<br>2. **我的学习轨迹**：记录查询与学习历史，并推荐知识缺口。<br>3. **积分与成就**：展示由贡献获得的积分、排行榜及数字勋章。 | 进化与治理中心（度量系统） |
| **团队管理后台** | 1. **成员与权限管理**：管理团队空间成员及权限（继承自主权限模型）。<br>2. **团队数据看板**：查看团队知识活跃度、贡献度排行等轻度分析数据。 | 安全与权限中心 |

### **四、 非功能性需求**

1.  **性能体验**：
    *   对话式问答响应时间 95% 以上请求 < 2秒。
    *   页面打开及核心操作（如保存、搜索）响应时间 < 1秒。
    *   支持千人规模团队并发使用，操作流畅无卡顿。
2.  **用户体验**：
    *   **界面设计**：现代、简洁、专业，符合企业级工具审美。
    *   **交互流畅**：所有交互流程需经过精心设计，符合直觉，学习成本低。
    *   **智能化感知**：AI辅助功能需精准、及时，减少用户错误操作和等待感。
3.  **兼容性**：
    *   支持主流浏览器（Chrome, Safari, Edge）最新版本。
    *   提供桌面端（Electron）及移动端适配，保障核心体验一致。
    *   与“无影”套件深度集成，保障第三方嵌入体验流畅。
4.  **可靠性**：
    *   前端应用自身可用性 ≥ 99.5%。
    *   具备优雅降级能力，当部分“灵枢”服务异常时，前端应有明确提示并尽可能保持基础功能可用。

---
**文档说明**：本文档定义了“智核”智能工作台作为独立前端产品的全部需求。其所有智能能力均依赖于与“灵枢”平台的稳定、高效对接。与“无影”套件的协同，主要体现在“情境知识面板”等功能对第三方系统的嵌入支持上。
