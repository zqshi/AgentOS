# 「智核」产品顶层设计

**版本**：V1.0  
**定位**：企业全员智能工作台与灵枢能力的前台交付层  
**适用范围**：智核产品方向（前台智能工作台）

---

## 1. 产品目标

- 提供统一的智能交互入口（对话、搜索、情境面板）。
- 让知识生产与消费自然融入工作流程。
- 将灵枢能力前台化、情境化与可视化。

---

## 2. 核心能力与模块划分

1. **主工作台**：对话、搜索、任务编排与摘要。  
2. **智能知识库**：多视图浏览、AI创作、版本与评审。  
3. **情境知识面板**：跨系统伴随式知识推送。  
4. **团队空间与项目协作**：团队知识聚合与协作沉淀。  
5. **智能体市场与应用**：发现、订阅与配置。  
6. **协作与审阅工作流**：评审、发布、意见沉淀。  
7. **个人中心与管理后台**：偏好、权限、团队管理。

---

## 3. 关键用户与场景

- **全员用户**：问答检索、任务协同、情境提示。  
- **领域专家**：知识创作、评审与协作。  
- **团队负责人**：团队空间与知识治理。  
- **管理者**：洞察与决策支持。

---

## 4. 关键业务流程（顶层）

1. **对话入口**：自然语言请求 → 调用灵枢能力 → 结果反馈。  
2. **知识生产**：创作 → AI辅助 → 评审 → 发布 → 入图谱。  
3. **情境伴随**：识别场景 → 推送知识 → 回写反馈。  
4. **团队协作**：空间组织 → 沉淀知识 → 贡献度量。

---

## 5. 与其他产品的边界

- **对灵枢**：智核仅做前台交付，不承载核心知识计算。  
- **对无影**：使用无影嵌入能力拓展情境面板覆盖面。  
- **自身边界**：智核不替代IM或项目管理系统。

---

## 6. 价值度量（方向性）

- 日活与使用频次。  
- 查询命中率与任务完成效率。  
- 知识贡献度与协作活跃度。  
- 智能体使用率与满意度。
