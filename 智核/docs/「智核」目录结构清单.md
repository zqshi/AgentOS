# 「智核」目录结构清单

**版本**：V1.0  
**适用范围**：智核产品方向（前台智能工作台）

---

## 1. 顶层结构

```
智核/
├── apps/                 # 可执行应用
├── docs/                 # 智核文档
├── packages/             # 智核内复用模块
├── scripts/              # 构建/测试/发布脚本
└── zhi-he/               # 现有前端原型/站点
```

---

## 2. apps 结构说明

```
apps/
└── core-service/         # 核心后端服务占位骨架
    ├── src/
    │   ├── contexts/     # DDD 有界上下文
    │   │   ├── workbench/
    │   │   ├── knowledge-base/
    │   │   ├── context-panel/
    │   │   ├── team-space/
    │   │   ├── agent-market/
    │   │   ├── collaboration/
    │   │   └── settings/
    │   ├── domain/
    │   ├── application/
    │   ├── infrastructure/
    │   └── interfaces/
    ├── tests/
    └── README.md
```

---

## 3. 上下文说明（对应业务模块）

- **workbench**：对话、任务编排、搜索与摘要  
- **knowledge-base**：文档管理、版本、评审与图谱视图  
- **context-panel**：情境识别、嵌入面板与快捷操作  
- **team-space**：团队空间与知识沉淀  
- **agent-market**：智能体发现、订阅与配置  
- **collaboration**：评审流程与评论协作  
- **settings**：个人偏好与团队管理
