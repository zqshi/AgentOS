## **「智核」平台 - "轻量生产工具"模块详细产品需求文档**

**文档编号**: ZH-PRD-M8
**模块名称**: 轻量生产工具
**产品定位**: 无摩擦知识捕获系统,让知识生产随时随地、随手可得。
**核心目标**: 通过极致轻量化的工具和交互方式,将知识捕获的门槛降到最低,让员工在灵感闪现、信息发现的第一时间完成记录,避免知识流失,并自动汇入企业知识网络。

---

## **产品边界澄清：轻量工具不是什么**

### **轻量工具 ≠ 会议系统**
- **不替代** Zoom/Teams/腾讯会议/钉钉会议
- **不提供** 视频会议、会议室管理、会议调度、屏幕共享
- **不做** 多人实时通话、会议主持控制

### **轻量工具 ≠ 笔记应用**
- **不替代** Notion/印象笔记/Obsidian等完整笔记系统
- **不提供** 复杂的文档编辑、数据库、双向链接
- **不做** 长篇文档创作（应使用智核知识库模块）

### **轻量工具 = 知识捕获入口**
- **是** 多端、多模态的快速记录工具
- **是** 音频/视频/图片/文本的**捕获与转写**工具
- **是** 将碎片化信息快速存入智核的**入口**
- **捕获后** 内容自动提交到「灵枢」加工，转化为结构化知识

### **关键区别：捕获 vs 管理**

| 场景 | 传统做法 | 轻量工具的职责 | 智核其他模块的职责 |
|:---|:---|:---|:---|
| **会议场景** | 使用Zoom开会 | **录音转文字**：录制音频并实时转写<br>**不做**：不提供视频会议能力 | **会议纪要**：团队空间模块接收录音数据，AI生成会议纪要并归档 |
| **文档创作** | 使用Word/飞书文档 | **速记草稿**：快速记录灵感和要点<br>**不做**：不做长文档编辑 | **正式文档**：知识库模块提供完整编辑器 |
| **网页收藏** | 使用浏览器书签 | **剪藏保存**：保存网页内容+自动提取<br>**不做**：不做阅读管理 | **知识入库**：灵枢加工后自动分类和关联 |
| **代码记录** | 复制到笔记本 | **代码片段捕获**：一键保存+自动标注<br>**不做**：不做代码管理 | **技术知识库**：知识库模块归档和关联 |

**核心原则**：轻量工具只做**捕获**，不做**管理**。捕获的内容提交到「灵枢」加工后，由智核的其他模块（知识库、团队空间）进行管理和呈现。

### **一、模块全景与核心价值**

#### **1.1 要解决的根本问题**

企业知识生产面临"最后一公里"困境:

1. **灵感易逝**:员工在通勤、会议、阅读中产生想法或发现有价值信息,但等到回到电脑前打开知识库时,灵感已忘记或信息已找不到。
2. **捕获门槛高**:传统知识记录需要打开应用、创建文档、填写标题、选择分类,步骤繁琐,员工懒得记录,导致大量隐性知识流失。
3. **场景割裂**:工作场景高度碎片化(手机、平板、电脑、网页),但知识捕获工具往往只在某一端好用,跨设备体验断裂。

轻量生产工具模块通过**多端覆盖**、**一键捕获**、**智能加工**三大能力,将知识捕获从"任务"变为"习惯"。

#### **1.2 核心设计理念**

1. **极致快速 (Speed First)**:从产生记录想法到完成捕获,不超过10秒;从发现信息到保存,不超过3次点击。
2. **智能补全 (AI-Powered)**:用户只需提供原始信息,系统自动补全标题、摘要、分类、标签,降低思考负担。
3. **无缝衔接 (Seamless Integration)**:捕获的内容自动进入「灵枢」知识熔炉加工队列,无需手动整理即可融入知识网络。
4. **多端一致 (Cross-Platform)**:Web、移动端、桌面端、浏览器插件体验一致,随时随地可用。

#### **1.3 核心用户角色**

| 角色 | 核心使用场景 | 关键价值获得 |
| :--- | :--- | :--- |
| **移动办公人员** | 通勤路上、客户现场记录灵感和待办;拍照记录现场情况;录音记录会议要点。 | 随时随地捕获,不遗漏任何想法;语音输入解放双手;自动转文字和提取摘要。 |
| **知识工作者** | 浏览网页时发现有价值内容需要保存;阅读PDF时标注重点;观看视频时记录关键信息。 | 一键剪藏不打断阅读流;自动提取关键信息;智能关联相关知识。 |
| **研发人员** | 在IDE中遇到有价值的代码片段或技术方案需要记录;看到好的开源项目需要收藏。 | 代码片段自动格式化;技术方案自动分类;关联项目和技术栈。 |
| **管理者** | 会议中快速记录决策和待办;现场巡视拍照记录问题;听汇报时录音备忘。 | 快速记录不影响会议节奏;自动生成会议纪要;智能提取任务和跟进项。 |
| **所有员工** | 看到好的文章/视频/图片想分享给团队;突发奇想有个点子想记下;看到竞品动态需要记录。 | 极低门槛记录和分享;系统智能整理;知识自动沉淀到企业资产。 |

### **二、详细功能需求**

#### **M8.1 移动端快速捕获（智核App）**

**目标**:让员工的手机成为最强大的知识捕获设备,支持文字、语音、图片、视频等多模态快速输入。

| 功能组件 | 详细需求描述 |
| :--- | :--- |
| **1. 快速笔记(速记)** | • **极速启动**:<br>  - App首屏提供显眼的"速记"按钮(固定底部中央)。<br>  - 支持3D Touch/长按App图标唤出"快速速记"快捷方式,无需打开App即可开始记录。<br>  - 支持iOS/Android系统级分享菜单,任何App中都可"分享到智核速记"。<br>• **多模态输入**:<br>  - **文字输入**:打开即可输入,支持富文本(标题、列表、加粗)。<br>  - **语音输入**:长按语音按钮说话,松手自动转文字(支持实时预览),基于「灵枢」多模态能力实现高准确率。<br>  - **拍照/相册**:一键拍照或选择相册图片,AI自动识别图片中的文字(OCR)、对象、场景。<br>  - **扫描文档**:相机扫描纸质文档,自动矫正透视、增强对比度、提取文字。<br>  - **手写识别**:支持手写输入(平板触控笔优化),自动转文字或保留手写原图。<br>• **智能辅助**:<br>  - **AI标题生成**:用户只需输入内容,AI自动生成合适的标题(可编辑)。<br>  - **自动分类**:AI根据内容自动判断类型(灵感、待办、会议记录、资料摘录),建议分类标签。<br>  - **关联推荐**:实时分析内容,推荐关联的项目、主题、标签、相关人员(可一键添加)。<br>• **快速保存**:<br>  - 点击"保存"立即完成(无需填写复杂表单),内容进入"待处理队列"。<br>  - 支持"保存并分享"到指定团队空间或@特定同事。<br>  - 支持设置提醒("明天9点提醒我处理这条笔记")。 |
| **2. 音频/视频捕获与转写** | • **长音频录制与转写**（适用于会议、访谈、培训等场景）：<br>  - 一键开启"录音",持续录音支持长达2小时。<br>  - **实时转写**:边录音边显示文字稿(带时间戳),支持暂停/继续。<br>  - **说话人分离**:AI自动识别不同说话人(如"张三:...","李四:...")。<br>  - **基础信息提取**（仅做初步加工）:<br>    &nbsp;&nbsp;• 自动生成简要摘要(提取关键词和主题)。<br>    &nbsp;&nbsp;• 标记可能的决策点和任务（待「灵枢」深度加工）。<br>    &nbsp;&nbsp;• 关键时刻标记(如"讨论激烈片段"),点击可跳转播放。<br>  - **数据流向**:录音+转写文本提交到「灵枢」知识熔炉 → 团队空间模块接收并生成正式会议纪要。<br>  - **明确边界**:不提供会议调度、参会人管理、会议室预定等会议管理功能。<br>• **短视频捕获**（用于现场记录）:<br>  - 录制短视频(最长5分钟),用于记录现场情况(如产品演示、设备故障、工地巡查)。<br>  - AI自动生成视频封面、提取关键帧、识别视频中的文字和对象。<br>  - 视频摘要:AI生成"本视频讲了什么"的文字说明。<br>• **语音备忘**（快速口述）:<br>  - 快速录制30秒语音备忘(如"提醒自己明天跟进张总的需求"),系统自动转文字并设置提醒。<br>  - 适用于走路、开车时的快速记录。 |
| **3. 一键分享捕获** | • **系统分享菜单集成**:<br>  - 在其他App(微信、浏览器、邮件、PDF阅读器)中,通过"分享"功能直接发送到智核。<br>  - 支持内容类型:<br>    &nbsp;&nbsp;• 网页链接(自动提取标题、摘要、正文)。<br>    &nbsp;&nbsp;• 微信文章(保存完整内容,避免失效)。<br>    &nbsp;&nbsp;• 图片(批量保存,AI自动分类)。<br>    &nbsp;&nbsp;• PDF文件(自动上传并提取目录和摘要)。<br>    &nbsp;&nbsp;• 纯文本(如复制的一段话)。<br>• **智能解析与标注**:<br>  - 保存时可快速添加注释("为什么保存这个?"),AI自动关联上下文。<br>  - AI判断内容价值度(高/中/低),优先推送高价值内容给相关人员。 |
| **4. 待处理队列与管理** | • **我的速记列表**:<br>  - 展示所有捕获的内容,按时间倒序排列。<br>  - 状态标识:草稿(橙色) / 加工中(蓝色) / 已入库(绿色)。<br>  - 支持筛选(按类型/日期/状态)和搜索。<br>• **批量整理**:<br>  - 选择多条速记,批量操作:<br>    &nbsp;&nbsp;• 合并为一篇文档(如将会议中的多条笔记合并为完整会议记录)。<br>    &nbsp;&nbsp;• 批量添加标签。<br>    &nbsp;&nbsp;• 批量分享到团队。<br>    &nbsp;&nbsp;• 批量删除。<br>• **加工状态追踪**:<br>  - 查看速记的「灵枢」加工进度:"已提取3个实体,建立了2个关联,等待审核"。<br>  - 加工完成后推送通知,可跳转查看最终入库的知识卡片或文档。 |

#### **M8.2 浏览器快速剪藏插件**

**目标**:让Web浏览成为知识生产的富矿,一键捕获网页内容,自动转化为结构化知识。

| 功能组件 | 详细需求描述 |
| :--- | :--- |
| **1. 全页/区域剪藏** | • **一键全页保存**:<br>  - 点击插件图标 → "保存整页",完整保存网页为:<br>    &nbsp;&nbsp;• **纯文本模式**:提取正文,去除广告和无关元素,保存为Markdown。<br>    &nbsp;&nbsp;• **阅读模式**:类似Safari阅读器,保留排版和图片,去除干扰元素。<br>    &nbsp;&nbsp;• **完整网页**:保存HTML+CSS+图片,离线可访问(应对链接失效)。<br>    &nbsp;&nbsp;• **PDF快照**:渲染为PDF,保留原始外观。<br>  - 自动提取元信息:标题、作者、发布时间、来源网站、摘要。<br>• **智能区域选择**:<br>  - 点击"选择区域",鼠标框选网页中感兴趣的部分(如某个段落、表格、图表)。<br>  - 所见即所得保存,保留格式和图片。<br>• **截图标注**:<br>  - 点击"截图",框选区域后进入标注模式:<br>    &nbsp;&nbsp;• 支持箭头、文字、高亮、马赛克。<br>    &nbsp;&nbsp;• 保存为带标注的图片+文字说明。 |
| **2. 划词速记** | • **智能划词工具栏**:<br>  - 在任何网页选中文本,自动弹出浮动工具栏(不遮挡内容)。<br>  - 工具栏按钮:<br>    &nbsp;&nbsp;• **速记**:将选中文本保存为速记卡片,可添加个人注释。<br>    &nbsp;&nbsp;• **解释**:调用「智核」本体库,解释专业术语(如功能清单M3中的划词取义)。<br>    &nbsp;&nbsp;• **搜索**:在企业知识库中搜索相关内容。<br>    &nbsp;&nbsp;• **翻译**:翻译外文内容(集成翻译服务)。<br>• **高亮与笔记**:<br>  - 选中文本后选择颜色高亮(黄色/绿色/粉色),下次访问该网页时高亮保留。<br>  - 可为高亮添加笔记气泡,鼠标悬停显示。<br>  - 所有高亮和笔记同步到云端,跨设备可见。 |
| **3. 智能内容解析** | • **文章智能提取**:<br>  - AI自动识别文章类型(技术文档/市场报告/新闻/教程),应用不同解析策略。<br>  - 提取关键信息:<br>    &nbsp;&nbsp;• **技术文章**:提取代码片段、API列表、技术栈。<br>    &nbsp;&nbsp;• **市场报告**:提取关键数据、图表、结论。<br>    &nbsp;&nbsp;• **新闻**:提取5W1H(谁、什么、何时、何地、为何、如何)。<br>    &nbsp;&nbsp;• **产品介绍**:提取产品名、功能列表、定价、竞品对比。<br>• **多媒体内容处理**:<br>  - **视频**:保存YouTube/B站等视频链接,AI提取视频字幕、生成摘要、提取关键帧。<br>  - **播客**:保存播客链接和文字稿(如有)。<br>  - **PPT/PDF**:保存SlideShare/Scribd等平台的演示文稿,自动提取目录和关键页。<br>• **自动关联与分类**:<br>  - 保存时,AI自动:<br>    &nbsp;&nbsp;• 判断内容主题,推荐标签和分类(如"机器学习"、"竞品分析"、"客户案例")。<br>    &nbsp;&nbsp;• 识别内容中的实体(公司、人物、产品、技术),自动关联到知识图谱。<br>    &nbsp;&nbsp;• 推荐相关的已有知识("您可能还想看这3篇相关文档")。 |
| **4. 稍后阅读与管理** | • **稍后阅读列表**:<br>  - 快速将网页加入"稍后阅读",无需详细标注(用于暂存感兴趣但没时间细看的内容)。<br>  - 稍后阅读列表在App和Web端同步,支持离线阅读。<br>  - 阅读完成后可一键"转为正式知识"或删除。<br>• **阅读统计**:<br>  - 记录保存了多少网页、阅读了多少、完成率。<br>  - 推荐"您保存但未读的高价值内容"。 |

#### **M8.3 桌面快捷工具(Desktop Client)** **目标**:在电脑工作时,提供全局可用的快速捕获能力,不打断工作流。

| 功能组件 | 详细需求描述 |
| :--- | :--- |
| **1. 全局悬浮窗** | • **常驻悬浮球**:<br>  - 桌面右下角显示半透明悬浮球(可拖动位置)。<br>  - 鼠标悬停展开快捷菜单:<br>    &nbsp;&nbsp;• 快速笔记<br>    &nbsp;&nbsp;• 截图<br>    &nbsp;&nbsp;• 录屏<br>    &nbsp;&nbsp;• 语音备忘<br>    &nbsp;&nbsp;• 打开智核<br>• **全局快捷键**:<br>  - 设置全局快捷键(如Ctrl+Shift+N),在任何应用中快速唤出速记窗口。<br>  - 速记窗口以浮窗形式出现,不打断当前工作。<br>  - 输入完成后自动隐藏,内容已保存。 |
| **2. 智能截图与标注** | • **多模式截图**:<br>  - **区域截图**:框选任意区域截图。<br>  - **窗口截图**:自动识别活动窗口,一键截取完整窗口(含阴影)。<br>  - **滚动截图**:自动滚动页面,截取长网页/长文档完整内容。<br>• **即时标注**:<br>  - 截图后立即进入标注编辑器:<br>    &nbsp;&nbsp;• 绘制工具:矩形、椭圆、箭头、画笔、文字、序号标记。<br>    &nbsp;&nbsp;• 模糊/马赛克(遮盖敏感信息)。<br>    &nbsp;&nbsp;• 撤销/重做、图层管理。<br>  - 标注完成后操作:<br>    &nbsp;&nbsp;• **保存到智核**:作为知识卡片保存,可添加说明。<br>    &nbsp;&nbsp;• **复制到剪贴板**:快速粘贴到其他应用。<br>    &nbsp;&nbsp;• **分享给同事**:生成链接或直接@发送。<br>• **OCR文字提取**:<br>  - 截图后自动识别图片中的文字,点击"提取文字"即可复制或编辑。<br>  - 支持中英文混排、公式识别(基于「灵枢」多模态能力)。 |
| **3. 录屏与屏幕录制** | • **快速录屏**:<br>  - 选择录制区域(全屏/指定窗口/自定义区域)。<br>  - 一键开始录制,悬浮控制条显示录制时长,支持暂停/继续/结束。<br>  - 可选是否录制系统声音、麦克风声音。<br>• **视频标注**:<br>  - 录制过程中可实时标注(高亮光标、绘制箭头),增强演示效果。<br>• **智能后处理**:<br>  - 录制完成后自动:<br>    &nbsp;&nbsp;• 压缩视频(可配置清晰度)。<br>    &nbsp;&nbsp;• 生成缩略图和封面。<br>    &nbsp;&nbsp;• 提取关键帧(每隔5秒一帧)。<br>  - AI生成视频说明:"本视频演示了XX功能的操作流程,共3个步骤"。<br>• **用途场景**:<br>  - 操作教程录制(分享给同事或新人培训)。<br>  - Bug复现录制(提交给研发)。<br>  - 产品演示录制(分享给客户)。 |
| **4. 剪贴板历史与管理** | • **增强剪贴板**:<br>  - 自动记录所有复制内容(文本/图片/文件),保留100条历史。<br>  - 快捷键调出剪贴板历史面板,搜索和快速粘贴历史内容。<br>• **智能剪贴板**:<br>  - AI自动分类剪贴板内容(代码/链接/文本/数据)。<br>  - 代码片段自动识别语言并语法高亮。<br>  - 链接自动提取标题和摘要。<br>• **一键入库**:<br>  - 剪贴板历史中任意内容可一键"保存到智核",自动转为知识卡片。 |

**目标**:让研发人员在编码时无缝捕获技术知识,将代码片段、技术方案、踩坑经验快速沉淀。

### **三、后台智能加工流程**

#### **3.1 捕获内容的自动加工**

所有通过轻量工具捕获的内容,自动进入「灵枢-知识熔炉」的加工流程:

| 加工步骤 | 处理内容 |
| :--- | :--- |
| **1. 内容理解与提取** | • 文本:分词、实体识别(人名、地名、机构、术语)、关键词提取、情感分析。<br>• 图片:OCR文字识别、对象识别、场景识别。<br>• 音频:语音转文字、说话人分离、关键段落提取。<br>• 视频:关键帧提取、字幕识别、场景切分。 |
| **2. 结构化转换** | • 非结构化内容转为结构化数据:<br>  - 录音数据 → 基础转写稿（**深度加工由团队空间模块完成**：会议记录、摘要、决策、任务）。<br>  - 网页剪藏 → 知识卡片(标题+摘要+正文+来源+关联实体)。<br>  - 代码片段 → 技术卡片(代码+说明+技术栈+适用场景)。<br>  - 视频捕获 → 视频元数据+关键帧+字幕(如有)。 |
| **3. 知识图谱关联** | • 自动将提取的实体关联到知识图谱:<br>  - 识别到"张三",关联到员工实体"张三@研发部"。<br>  - 识别到"产品A",关联到产品实体及其属性。<br>  - 识别到"机器学习",关联到技术领域本体。<br>• 建立内容间的关联:<br>  - "本次会议讨论的是上周决策的后续执行",自动关联两次会议记录。 |
| **4. 质量评估与分类** | • AI评估内容质量和价值:<br>  - **高价值**:包含独特见解、重要决策、稀缺信息,自动推荐给相关人员。<br>  - **中等价值**:常规知识,正常入库。<br>  - **低价值**:临时备忘、冗余信息,进入个人草稿区,不污染公共知识库。<br>• 自动分类到合适的知识领域和主题。 |
| **5. 推送与通知** | • 根据内容相关性,智能推送:<br>  - 涉及某个项目的内容,自动关联到项目空间,通知项目成员。<br>  - 提到某个同事,自动@通知该同事。<br>  - 识别出任务,自动添加到任务列表或@负责人。 |

### **四、跨端体验一致性**

#### **4.1 数据实时同步**

- 所有端捕获的内容实时同步到云端,任意设备可访问。
- 支持离线捕获(如在飞机上),联网后自动上传和加工。

#### **4.2 统一的"待处理中心"**

- Web端、移动端、桌面端共享同一个"待处理队列"。
- 可在任意端查看、编辑、整理捕获的内容。
- 支持跨端协作(如手机捕获,电脑精编)。

#### **4.3 个性化配置同步**

- 用户的偏好设置(默认分类、常用标签、快捷键等)在所有端同步。

### **五、非功能需求**

#### **5.1 性能要求**
- 移动端速记启动时间 < 1秒。
- 浏览器插件剪藏响应时间 < 2秒。
- 桌面截图工具调用延迟 < 0.5秒。
- 语音转文字延迟 < 3秒(10秒语音)。

#### **5.2 可靠性要求**
- 捕获内容先保存本地,上传失败自动重试,确保零丢失。
- 支持离线捕获,联网后自动同步。
- 异常情况(如App崩溃)自动恢复草稿。

#### **5.3 易用性要求**
- 新用户首次使用速记功能,无需教程即可完成一次捕获。
- 从产生捕获想法到完成保存,全流程耗时 < 10秒。
- 全平台操作逻辑一致,学习一次处处适用。

#### **5.4 隐私与安全**
- 本地捕获的敏感内容(如财务数据、密码),不自动上传云端,需用户明确授权。
- 截图中的敏感信息(如身份证号、银行卡号)AI自动识别并提示模糊处理。
- 麦克风、相机权限需用户明确授权,使用时显示指示灯。

### **八、核心交互流程示例**

#### **场景1:移动端音频捕获（会议场景）**

**典型流程**：

1. **会前**：打开智核App，点击"录音"功能
2. **会中**：点击录音按钮开始，手机放桌上，全程录音（40分钟）
   - 实时显示转写文字稿（带时间戳）
   - 李总说到重要内容时，手动点击"标记关键点"
3. **会后**：停止录音，内容自动保存到"待处理队列"
   - 轻量工具完成的初步处理（< 1分钟）：
     - ✅ 语音转文字（完整转写稿）
     - ✅ 说话人分离标记
     - ✅ 关键时刻标记（3个）
     - ✅ 简要主题识别："产品规划会议"
4. **数据提交**：录音+转写稿自动提交到「灵枢」知识熔炉
5. **深度加工**（团队空间模块接管，需5-10分钟）：
   - 🔄 AI生成会议摘要（200字）
   - 🔄 提取决策事项（3条）
   - 🔄 提取待办任务（5条，自动@负责人）
   - 🔄 关联到"Q1产品规划"项目空间
6. **完成通知**：用户收到通知"会议纪要已生成"，可在团队空间查看正式会议纪要

**关键点**：
- **轻量工具的职责**：录音 + 转文字 + 基础标记（快速完成）
- **团队空间的职责**：深度理解 + 结构化提取 + 归档管理（需时较长但更准确）
- **用户价值**：从会议结束到获得正式纪要，全程10分钟（传统方式需1-2小时手动整理）

#### **场景2:浏览器剪藏（网页收藏）**

1. **用户**：在浏览器阅读技术文章时，发现有价值内容
2. **操作**：点击浏览器插件图标 → "保存整页"
3. **轻量工具**：
   - 提取网页正文（去广告、去干扰）
   - 自动识别标题、作者、发布时间
   - AI生成摘要和提取关键词
   - 用户可快速添加注释："待研究的新技术方案"
4. **提交**：内容提交到「灵枢」加工队列
5. **深度加工**：
   - 实体抽取（识别技术名词、公司、人物）
   - 关联到知识图谱（如"React"、"前端架构"）
   - 推荐相关已有文档
6. **入库**：加工完成后，自动归入知识库，用户收到通知

**用户价值**：从"看到好文章"到"收藏完成"，全程3秒，无需打断阅读。
