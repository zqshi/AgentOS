### **「智核」平台 - "情境知识面板"模块详细产品需求文档**

**文档编号**:ZH-PRD-M3
**模块名称**:情境知识面板
**产品定位**:无处不在的智能认知助手,将企业知识能力嵌入用户工作的每一个场景。
**核心目标**:通过轻量级、嵌入式的交互界面,在用户使用各类业务系统时,主动提供精准的知识支持与智能辅助,实现"知识随处可得"的理想体验。

---

### **一、 模块全景与核心价值**

#### **1.1 要解决的根本问题**
当前企业员工的工作场景高度分散,面临三大困境:
1. **场景切换成本高**:在CRM、代码编辑器、项目管理工具中工作时,需要切换到知识库查找信息,打断工作流。
2. **上下文丢失**:传统搜索无法理解用户当前的工作情境,返回泛化结果,需要人工筛选。
3. **知识孤岛**:业务系统与知识系统割裂,系统中的客户、项目、订单等数据,无法与企业知识网络关联。

#### **1.2 核心设计理念**
1. **嵌入式设计**:不要求用户离开工作场景,而是将知识能力"送"到用户眼前。
2. **情境感知优先**:深度理解用户当前的工作对象(客户、项目、代码文件等)和操作意图,提供精准服务。
3. **轻量交互**:通过浮动窗口、侧边栏、划词等轻量形式呈现,不干扰主工作界面。
4. **双向连接**:既从业务系统拉取上下文调用知识,也能将知识快速捕获回流到"灵枢"平台。

#### **1.3 核心用户角色**
| 角色 | 职责与目标 | 核心使用场景 |
| :--- | :--- | :--- |
| **业务人员** | 在CRM、ERP等系统中处理客户、订单时需要快速查阅相关知识。 | 在CRM查看客户时,调阅客户历史案例、风险提示。 |
| **研发人员** | 在代码编辑器中开发时需要查阅技术文档、API说明。 | 在IDE中查看某API时,调阅使用文档、代码示例。 |
| **知识工作者** | 在浏览网页、阅读文档时发现有价值信息,需要快速保存。 | 浏览竞品报告时,划词保存关键段落到知识库。 |
| **所有员工** | 在任何场景下遇到不理解的术语,需要即时解释。 | 在任何系统中划选专业术语,查看企业本体定义。 |

### **二、 详细功能需求**

#### **FR3.1 浏览器扩展插件**

**目标**:通过浏览器插件,将智能知识能力注入用户的Web工作场景。

| 功能组件 | 详细需求描述 |
| :--- | :--- |
| **1. 划词取义与知识查询** | • **智能划词识别**:<br>  &nbsp;&nbsp;- 用户在任何网页或企业Web应用中选中文本。<br>  &nbsp;&nbsp;- 选中后出现悬浮工具栏,提供"解释"、"搜索"、"保存"等快捷操作。<br>• **本体术语解释**:<br>  &nbsp;&nbsp;- 点击"解释",系统识别选中文本是否为企业本体中定义的术语。<br>  &nbsp;&nbsp;- 若是,弹出气泡卡片,展示:<br>    &nbsp;&nbsp;&nbsp;&nbsp;• 术语标准定义。<br>    &nbsp;&nbsp;&nbsp;&nbsp;• 所属业务域。<br>    &nbsp;&nbsp;&nbsp;&nbsp;• 关联的关键关系与属性。<br>    &nbsp;&nbsp;&nbsp;&nbsp;• 点击"查看更多"可跳转知识图谱。<br>• **智能搜索**:<br>  &nbsp;&nbsp;- 点击"搜索",以选中文本为关键词,调用"灵枢"知识图谱进行搜索。<br>  &nbsp;&nbsp;- 在侧边面板展示搜索结果(相关文档、案例、数据)。<br>• **多语言支持**:支持划选外文内容时,自动翻译并匹配内部知识。 |
| **2. 情境智能伴读面板** | • **自动情境识别**:<br>  &nbsp;&nbsp;- 插件能识别用户当前所在的业务系统(通过URL模式匹配或DOM分析)。<br>  &nbsp;&nbsp;- 例如:识别到用户在Salesforce客户详情页,提取客户ID。<br>• **侧边知识面板**:<br>  &nbsp;&nbsp;- 点击插件图标,在浏览器右侧展开侧边面板。<br>  &nbsp;&nbsp;- 面板根据当前页面情境,自动展示:<br>    &nbsp;&nbsp;&nbsp;&nbsp;• **实体摘要**:当前客户/项目/订单的核心信息摘要。<br>    &nbsp;&nbsp;&nbsp;&nbsp;• **关联知识**:历史案例、合作记录、风险提示、最佳实践。<br>    &nbsp;&nbsp;&nbsp;&nbsp;• **关联人员**:负责人、相关专家、协作同事。<br>    &nbsp;&nbsp;&nbsp;&nbsp;• **最近动态**:该实体的最新更新、讨论、任务。<br>• **快速操作**:<br>  &nbsp;&nbsp;- 面板底部提供快捷操作按钮(如"创建任务"、"发起会议"、"提交反馈")。 |
| **3. 快速捕获工具** | • **网页剪藏**:<br>  &nbsp;&nbsp;- 点击插件工具栏的"剪藏"按钮,支持:<br>    &nbsp;&nbsp;&nbsp;&nbsp;• **全页保存**:保存完整网页为PDF或HTML。<br>    &nbsp;&nbsp;&nbsp;&nbsp;• **区域截图**:框选区域截图保存。<br>    &nbsp;&nbsp;&nbsp;&nbsp;• **文本摘录**:仅保存选中的文本段落。<br>• **智能标注**:<br>  &nbsp;&nbsp;- 保存时,插件自动:<br>    &nbsp;&nbsp;&nbsp;&nbsp;• 提取标题、摘要、关键词。<br>    &nbsp;&nbsp;&nbsp;&nbsp;• 识别内容类型(竞品分析/技术文档/市场报告)。<br>    &nbsp;&nbsp;&nbsp;&nbsp;• 建议关联的项目、主题、标签。<br>  &nbsp;&nbsp;- 用户可编辑标注信息,确认后保存。<br>• **入库流程**:<br>  &nbsp;&nbsp;- 保存的内容作为"知识卡片草稿"提交到"灵枢-知识熔炉"待处理队列。<br>  &nbsp;&nbsp;- 系统后台自动加工(实体抽取、关系建立),完成后通知用户。 |
| **4. 跨系统统一体验** | • **支持的系统**:<br>  &nbsp;&nbsp;- 优先支持企业高频使用的Web系统(CRM、项目管理、文档协作)。<br>  &nbsp;&nbsp;- 提供配置界面,管理员可添加新的系统URL规则和情境提取逻辑。<br>• **统一快捷键**:<br>  &nbsp;&nbsp;- 全局快捷键(如Alt+K)在任何页面快速调出知识搜索框。<br>  &nbsp;&nbsp;- 全局快捷键(如Alt+S)快速保存当前页面或选中内容。 |

#### **FR3.2 桌面悬浮窗与快速入口**

**目标**:在用户使用本地应用(IDE、Office等)时,提供轻量级的知识接入能力。

| 功能组件 | 详细需求描述 |
| :--- | :--- |
| **1. 桌面悬浮球** | • **常驻悬浮**:<br>  &nbsp;&nbsp;- 在桌面提供可拖拽的半透明悬浮球(类似macOS的Spotlight)。<br>  &nbsp;&nbsp;- 悬浮球默认停靠在屏幕边缘,鼠标悬停或点击展开。<br>• **快速搜索**:<br>  &nbsp;&nbsp;- 点击悬浮球,弹出搜索框,输入关键词即时搜索知识库。<br>  &nbsp;&nbsp;- 支持快捷操作(如"新建笔记"、"查看今日摘要")。<br>• **剪贴板监听(可选)**:<br>  &nbsp;&nbsp;- 用户开启后,复制文本时,悬浮球自动检测是否为企业相关概念。<br>  &nbsp;&nbsp;- 若是,弹出提示气泡:"检测到术语'XX',点击查看定义"。 |
| **2. IDE集成插件** | • **支持的IDE**:优先支持VS Code、JetBrains系列(IntelliJ、PyCharm)。<br>• **代码情境感知**:<br>  &nbsp;&nbsp;- 识别用户当前打开的代码文件、函数、类名。<br>  &nbsp;&nbsp;- 在IDE侧边栏展示关联的技术文档、代码注释、历史变更说明。<br>• **API文档即时查看**:<br>  &nbsp;&nbsp;- 光标停留在API调用处,自动展示该API的文档(参数说明、返回值、示例代码)。<br>  &nbsp;&nbsp;- 文档来源于"灵枢"知识图谱中维护的技术知识。<br>• **代码片段保存**:<br>  &nbsp;&nbsp;- 选中代码片段,右键"保存为知识卡片",添加说明后保存到知识库。 |
| **3. Office套件插件** | • **Word/Excel/PPT插件**:<br>  &nbsp;&nbsp;- 在Office工具栏添加"智核助手"插件按钮。<br>  &nbsp;&nbsp;- 点击后在侧边栏打开知识面板,提供:<br>    &nbsp;&nbsp;&nbsp;&nbsp;• 术语查询与插入。<br>    &nbsp;&nbsp;&nbsp;&nbsp;• 内容推荐(如写PPT时推荐相关案例、数据图表)。<br>    &nbsp;&nbsp;&nbsp;&nbsp;• 文档规范检查(如检测术语一致性)。<br>• **智能引用**:<br>  &nbsp;&nbsp;- 从知识面板拖拽内容到文档中,自动插入并保留来源引用。 |

#### **FR3.3 移动端轻量化支持**

**目标**:在移动场景下,提供核心的知识查询与捕获能力。

| 功能组件 | 详细需求描述 |
| :--- | :--- |
| **1. 移动端小组件** | • **iOS/Android桌面小组件**:<br>  &nbsp;&nbsp;- 提供快捷搜索框小组件,点击直接进入搜索。<br>  &nbsp;&nbsp;- 提供每日摘要小组件,展示今日待办与知识推荐。<br>• **快捷指令/Siri集成**:<br>  &nbsp;&nbsp;- iOS用户可通过Siri语音搜索知识("嘿Siri,在智核搜索客户管理流程")。<br>  &nbsp;&nbsp;- Android用户可通过Google Assistant集成。 |
| **2. 分享扩展** | • **系统分享接入**:<br>  &nbsp;&nbsp;- 在移动端浏览器或其他应用中,点击"分享"按钮,选择"智核"。<br>  &nbsp;&nbsp;- 快速将内容(网页、图片、文本)保存为知识卡片草稿。<br>• **微信/企业微信集成**:<br>  &nbsp;&nbsp;- 支持在微信/企微中分享内容到"智核助手"小程序。<br>  &nbsp;&nbsp;- 小程序提供简化的标注与保存界面。 |

#### **FR3.4 智能提醒与主动推送**

**目标**:将被动的知识查询,转变为主动的情境感知推送。

| 功能组件 | 详细需求描述 |
| :--- | :--- |
| **1. 情境触发推送** | • **工作场景识别**:<br>  &nbsp;&nbsp;- 系统识别用户进入特定工作场景(如打开某个高风险客户的CRM页面)。<br>  &nbsp;&nbsp;- 自动推送通知:"该客户近期有投诉记录,建议查看处理方案"。<br>• **时间触发推送**:<br>  &nbsp;&nbsp;- 会议前15分钟,推送会议相关的背景资料、上次会议纪要。<br>  &nbsp;&nbsp;- 项目截止日前3天,推送项目关键文档与风险检查清单。 |
| **2. 智能提醒气泡** | • **轻量化提示**:<br>  &nbsp;&nbsp;- 以系统通知或浮动气泡形式展示提醒。<br>  &nbsp;&nbsp;- 用户可快速操作(如"查看详情"、"稍后提醒"、"标记已读")。<br>• **提醒管理**:<br>  &nbsp;&nbsp;- 用户可在设置中配置提醒偏好(频率、类型、勿扰时段)。 |

### **三、 核心业务流程示例**

**流程:销售人员在CRM中处理客户时获得智能知识支持**

1. **打开客户页面**:
   - 销售人员在Salesforce中打开客户"XX科技公司"的详情页。

2. **情境识别与面板展示**:
   - 浏览器插件自动识别当前页面为客户详情页,提取客户名称和ID。
   - 侧边面板自动展开,展示:
     - **客户摘要**:行业、规模、合作年限、当前合同状态。
     - **关联知识**:过往合作案例、成功经验、投诉记录、竞争对手动态。
     - **关键联系人**:该客户的对接人信息、我方负责团队。
     - **风险提示**:系统标注"该客户近期付款延迟,建议关注财务状况"。

3. **深入查询**:
   - 销售人员点击"投诉记录",跳转查看详细的投诉处理文档。
   - 在文档中划选"解决方案",查看该方案的效果与评价。

4. **快速行动**:
   - 销售人员在面板底部点击"发起会议",系统自动创建会议邀请,并关联客户与相关文档。

5. **知识反哺**:
   - 销售人员本次沟通发现了客户的新需求,通过悬浮球快速添加笔记。
   - 笔记自动关联到"XX科技公司"实体,丰富知识图谱。

### **五、 非功能性需求**

| 需求类别 | 具体要求 |
| :--- | :--- |
| **性能** | • 浏览器插件加载时间 < 500ms,不影响页面正常使用。<br>• 划词响应延迟 < 300ms。<br>• 情境面板内容加载 < 2秒。 |
| **兼容性** | • 浏览器插件支持Chrome、Edge、Safari最新版本。<br>• 桌面悬浮窗支持Windows 10+、macOS 11+。<br>• IDE插件支持VS Code 1.60+、IntelliJ 2021.1+。 |
| **资源占用** | • 浏览器插件内存占用 < 50MB。<br>• 桌面悬浮窗CPU占用率 < 1%,内存 < 100MB。 |
| **隐私与安全** | • 插件不主动收集用户浏览历史,仅在用户主动触发时请求知识。<br>• 剪贴板监听功能默认关闭,需用户手动开启并授权。<br>• 捕获的内容加密传输,遵循企业数据安全规范。 |
| **用户体验** | • 界面设计轻量、简洁,不干扰主工作界面。<br>• 支持自定义插件行为(如禁用特定页面、调整面板位置)。<br>• 提供详细的新手引导与使用文档。 |
