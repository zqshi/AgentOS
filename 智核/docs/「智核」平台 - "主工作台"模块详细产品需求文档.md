## **「智核」平台 - "主工作台"模块详细产品需求文档**

**文档编号**: ZH-PRD-M1
**模块名称**: 主工作台(智能交互中枢)
**产品定位**: 企业知识的智能交互门户，连接人与知识网络的对话式界面
**核心目标**: 让员工通过自然语言对话，零门槛地获取知识、完成任务、协同智能体，成为知识工作的统一入口

---

## **产品定位澄清：智核不是什么**

### **智核 ≠ IM工具**
- **不替代**企业微信/钉钉/飞书/Slack
- **不提供**实时聊天、群组、视频会议功能
- **不做**即时通讯的消息推送和已读回执

### **智核 ≠ 项目管理工具**
- **不替代**Jira/Asana/Monday
- **不提供**完整的项目规划、甘特图、资源调度
- **不做**项目管理的流程引擎

### **智核 = 知识智能层**
- **是**企业知识的统一查询入口
- **是**知识生产的智能助手
- **是**将散落在各系统的信息转化为结构化知识的中枢
- **通过集成**从IM/会议/邮件/业务系统**自动捕获**和**智能加工**知识

---

## **一、模块全景与核心价值**

### **1.1 要解决的根本问题**

企业知识工作面临的核心困境：

1. **知识查找困难**
   - 信息散落在文档系统、IM群聊、邮件、业务系统
   - 不知道"谁知道什么"、"答案在哪里"
   - 搜索返回大量结果但找不到准确答案

2. **知识使用门槛高**
   - 需要了解复杂的系统结构和导航逻辑
   - 新员工学习成本高，老员工也记不住所有路径
   - 同一个问题被重复提问和解答无数次

3. **知识与工作割裂**
   - 知识库是知识库，工作是工作，两者缺乏连接
   - 做决策时想不起查阅相关知识
   - 工作中产生的智慧没有及时沉淀

### **1.2 核心设计理念**

**1. 对话即界面 (Conversation as Interface)**
- 80%的知识需求通过自然语言对话即可满足
- 无需学习菜单、记忆路径，像问同事一样问系统

**2. 知识触手可及 (Knowledge at Fingertips)**
- 从"我要找什么"到"我得到答案"，全程10秒内
- 不仅返回文档链接，直接给出精炼答案

**3. 工作即沉淀 (Work as Knowledge)**
- 在主工作台完成的对话、任务、分析自动转化为知识资产
- 知识生产不是额外负担，而是工作的自然产物

**4. 智能体协同 (AI-Agent Collaboration)**
- 复杂任务自动分解，调度智能体团队协同完成
- 用户只需描述目标，系统负责执行路径

### **1.3 与其他系统的边界**

| 系统类型 | 职责边界 | 智核的集成方式 |
|:---|:---|:---|
| **IM工具**(企业微信/钉钉) | 实时通讯、群聊、消息推送 | **单向数据获取**：通过API/Webhook获取群聊中的知识片段（如决策讨论、问答）并加工入库；<br>**消息推送**：知识更新通知可推送到IM |
| **会议系统**(Zoom/Teams) | 音视频会议、屏幕共享 | **会议纪要获取**：获取会议录音/录屏/聊天记录，AI提取会议纪要、决策、待办；<br>**不做**：不提供会议调度和视频能力 |
| **邮件系统**(Exchange/Gmail) | 邮件收发、日程管理 | **邮件知识提取**：识别邮件中的重要信息（合同、客户反馈）并结构化；<br>**不做**：不提供邮件客户端 |
| **业务系统**(CRM/ERP) | 业务数据录入和流转 | **数据关联查询**：在智核对话中可查询业务数据（如"客户X的最新订单"）；<br>**情境面板**：在业务系统中浏览时，侧边显示相关知识 |
| **项目管理**(Jira) | 项目规划、任务分配、进度跟踪 | **任务状态同步**：展示项目待办，完成状态回写；<br>**不做**：不提供完整项目管理流程 |

**核心原则**：智核是"知识层"，通过集成从各系统**获取原始信息** → **智能加工** → **结构化存储** → **智能分发**

## **二、详细功能需求**

### **M1.1 智能对话引擎**

**目标**：提供接近人类专家的对话问答体验，成为员工的首席知识助手

| 功能组件 | 详细需求描述 |
|:---|:---|
| **1. 对话界面** | • **中央对话框**：主工作台中心位置的大型对话输入框<br>• **多模态输入**：<br>&nbsp;&nbsp;- 文本输入（支持多行、Markdown预览）<br>&nbsp;&nbsp;- 语音输入（实时转文字，方言支持）<br>&nbsp;&nbsp;- 图片上传（OCR识别、图表理解）<br>&nbsp;&nbsp;- 文件上传（PDF/Word解析提问）<br>• **智能提示**：<br>&nbsp;&nbsp;- 输入时显示历史类似问题<br>&nbsp;&nbsp;- 推荐快捷指令（"/搜索"、"/分析"、"/总结"）<br>&nbsp;&nbsp;- 根据当前工作情境推荐提问 |
| **2. 问答能力** | • **事实问答**：<br>&nbsp;&nbsp;- "Q1北区销售额多少？" → 直接返回数字+来源<br>&nbsp;&nbsp;- 调用知识图谱精确查询，返回结构化数据<br>• **概念解释**：<br>&nbsp;&nbsp;- "什么是敏捷开发？" → 返回企业本体定义+相关文档<br>&nbsp;&nbsp;- 区分"企业内部定义"和"通用定义"<br>• **寻人问答**：<br>&nbsp;&nbsp;- "谁负责XX项目？" → 返回人员卡片+联系方式<br>&nbsp;&nbsp;- "谁最懂机器学习？" → 基于知识图谱贡献度推荐专家<br>• **流程咨询**：<br>&nbsp;&nbsp;- "如何报销？" → 返回流程SOP+可视化流程图<br>• **对比分析**：<br>&nbsp;&nbsp;- "产品A和B的区别？" → 返回对比表格 |
| **3. 任务编排** | • **意图识别**：识别复杂任务指令<br>&nbsp;&nbsp;- "帮我准备下周董事会的市场分析报告"<br>• **任务分解**：<br>&nbsp;&nbsp;- AI自动拆解为子任务（收集数据 → 分析 → 生成报告）<br>&nbsp;&nbsp;- 以清单形式展示给用户确认<br>• **智能体调度**：<br>&nbsp;&nbsp;- 调用「灵枢-智能体工厂」中的专业智能体<br>&nbsp;&nbsp;- 并行执行子任务，实时显示进度<br>• **结果交付**：<br>&nbsp;&nbsp;- 生成可交付文档（报告、PPT大纲、数据表）<br>&nbsp;&nbsp;- 自动保存到知识库并关联项目 |
| **4. 多轮对话** | • **上下文记忆**：完整记住本次对话的所有信息<br>• **指代消解**：理解"它"、"那个"、"刚才提到的"等指代<br>• **追问细化**：支持连续追问深入细节<br>• **话题切换**：识别话题转移，保留旧话题可回溯<br>• **对话导出**：一键导出对话为文档或知识卡片 |
| **5. 答案质量保障** | • **来源标注**：每个答案标注知识来源（文档、作者、时间）<br>• **置信度评估**：AI评估答案可信度（高/中/低/不确定）<br>• **多来源对比**：当多个来源信息冲突时，并列展示并标注差异<br>• **人类兜底**：当AI无法回答时，推荐相关领域专家<br>• **反馈闭环**：用户可点赞/踩/补充，数据回流优化模型 |

---

### **M1.2 工作情境感知与主动推送**

**目标**：从"被动回答"进化到"主动服务"，让知识在需要时自动出现

| 功能组件 | 详细需求描述 |
|:---|:---|
| **1. 今日智能摘要** | • **生成时机**：每日首次登录时展示<br>• **内容模块**：<br>&nbsp;&nbsp;- **待办事项**：来自各系统的待审批/待回复任务（优先级排序）<br>&nbsp;&nbsp;- **知识更新**：关注项目/标签的新文档摘要<br>&nbsp;&nbsp;- **知识推荐**：基于角色和近期工作的相关知识<br>&nbsp;&nbsp;- **专家推荐**：可能需要咨询的领域专家<br>• **个性化配置**：用户可自定义关注维度 |
| **2. 情境智能面板** | • **情境识别**：根据用户当前操作识别工作情境<br>&nbsp;&nbsp;- 正在编辑"产品需求文档" → 推荐相关需求案例<br>&nbsp;&nbsp;- 正在查看"客户X资料" → 展示历史合作记录<br>• **侧边面板**：<br>&nbsp;&nbsp;- 关联知识（相关文档、历史案例）<br>&nbsp;&nbsp;- 关联人员（项目成员、领域专家）<br>&nbsp;&nbsp;- 快捷操作（发起讨论、创建任务）<br>• **跨系统情境**：<br>&nbsp;&nbsp;- 通过浏览器插件，在CRM/ERP等系统中也能显示情境面板 |
| **3. 智能提醒** | • **知识过期提醒**："您参与的文档《XX》已3个月未更新，是否需要审核？"<br>• **知识缺口提醒**："您的团队在XX领域尚无知识沉淀，是否创建？"<br>• **专家求助提醒**："有3位同事在询问您擅长的XX问题，是否查看？" |

---

### **M1.3 全局智能搜索**

**目标**：不仅是"搜索引擎"，更是"知识入口"和"答案引擎"

| 功能组件 | 详细需求描述 |
|:---|:---|
| **1. 搜索入口** | • **全局搜索框**：主工作台顶部，快捷键Ctrl/Cmd+K调出<br>• **搜索建议**：输入时实时展示历史搜索、热门搜索、智能补全<br>• **高级筛选**：按类型/时间/来源/标签快速筛选 |
| **2. 混合检索** | • **三路并行**：<br>&nbsp;&nbsp;- 关键词检索（倒排索引，毫秒级响应）<br>&nbsp;&nbsp;- 语义检索（向量相似度，理解同义词和相关概念）<br>&nbsp;&nbsp;- 图谱查询（关系遍历，发现隐含关联）<br>• **智能排序**：综合相关度、权威性、时效性、个性化偏好 |
| **3. 结果呈现** | • **分栏布局**：<br>&nbsp;&nbsp;- **最佳答案区**（顶部）：AI生成的直接答案<br>&nbsp;&nbsp;- **文档区**（左侧）：相关文档列表<br>&nbsp;&nbsp;- **人员区**（右侧上）：相关专家<br>&nbsp;&nbsp;- **实体区**（右侧下）：知识图谱实体<br>• **快速预览**：悬停显示内容摘要，无需跳转<br>• **快捷操作**：打开、收藏、分享、查看关联 |
| **4. 无结果处理** | • **智能建议**："未找到精确匹配，但以下内容可能相关..."<br>• **人类兜底**："您可以向[专家]提问"<br>• **知识缺口记录**：所有无结果查询记录到「灵枢-进化中心」，驱动知识生产 |

---

### **M1.4 知识沉淀与对话记录**

**目标**：让主工作台的交互自动转化为知识资产

| 功能组件 | 详细需求描述 |
|:---|:---|
| **1. 对话记录管理** | • **会话列表**：左侧边栏展示所有对话会话<br>• **会话命名**：AI自动根据内容生成会话标题<br>• **会话标签**：手动或自动打标签（项目/主题）<br>• **会话搜索**：在历史对话中搜索内容<br>• **会话分享**：生成链接分享给同事 |
| **2. 自动知识提取** | • **问答提取**：高质量的问答自动提取为FAQ<br>• **决策提取**：对话中的决策自动标记并关联项目<br>• **任务提取**：对话中提到的待办自动创建任务<br>• **实体提取**：提到的人/项目/产品自动关联到知识图谱 |
| **3. 手动沉淀** | • **标记重点**：用户可标记对话中的某段内容为"重要"<br>• **转为文档**：选中对话片段，一键转为正式文档<br>• **创建知识卡片**：对话内容快速创建知识卡片 |

## **三、与「灵枢」平台的集成**

主工作台是「智核」的前端界面，深度依赖「灵枢」平台的智能能力：

| 「灵枢」模块 | 集成点 | 数据流向 |
|:---|:---|:---|
| **动态知识图谱** | 问答查询、实体关联、关系遍历 | 主工作台 → 灵枢查询；灵枢 → 返回结构化答案 |
| **智能体工厂** | 任务编排、智能体调度、任务执行 | 主工作台识别任务 → 灵枢分解执行 → 返回结果 |
| **本体论工厂** | 术语解释、概念查询 | 主工作台 → 查询术语定义；灵枢 → 返回标准定义 |
| **知识熔炉** | 对话内容加工、知识提取 | 主工作台对话数据 → 灵枢加工入库 |
| **进化与治理中心** | 用户反馈、搜索日志、知识缺口 | 主工作台使用数据 → 灵枢分析优化 |

## **四、非功能需求**

### **4.1 性能要求**
- 对话响应时间 < 2秒（简单问答）
- 复杂任务分解展示 < 5秒
- 搜索结果返回 < 1秒（10万文档内）
- 支持并发1000+用户同时对话

### **4.2 易用性要求**
- 新用户首次使用，无需教程即可完成一次问答
- 语音识别准确率 > 95%（普通话）
- 答案准确率 > 85%（基于人工评测）

### **4.3 可用性要求**
- 系统可用性 > 99.5%
- 降级策略：「灵枢」服务异常时，至少保证基础关键词搜索可用

## **五、实现优先级**

- 基础对话问答（事实查询、概念解释）
- 全局搜索（关键词+语义）
- 对话记录管理

### **P1（第一次迭代）**
- 任务编排与智能体调度
- 今日智能摘要
- 多轮对话与上下文记忆

### **P2（第二次迭代）**
- 情境智能面板
- 自动知识提取
- 语音输入

### **P3（长期优化）**
- 图谱可视化查询
- 深度个性化推荐
- 多语言支持

## **六、成功指标**

### **使用频率指标**
- 日活用户 > 60%（全员）
- 人均每日问答次数 > 5次
- 主工作台成为工作日首个打开的应用（目标占比 > 40%）

### **效率提升指标**
- 平均问题解决时间从15分钟降低到2分钟（80%缩减）
- 重复问题减少70%（通过知识沉淀）
- 跨系统查询次数减少50%（统一入口）

### **质量指标**
- 答案准确率 > 85%（人工评测）
- 用户满意度 > 4.3/5
- 答案采纳率 > 75%（用户点赞比例）
